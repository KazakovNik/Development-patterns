# О шаблоне
Паттерн Flyweight позволяет уменьшить количество создаваемых объектов за счёт повторного использования существующих экземпляров. Вместо того, чтобы создавать большое число схожих объектов, создаётся ограниченное количество общих объектов, каждый из которых многократно применяется в разных ситуациях. Это особенно полезно, когда существует много похожих объектов, но различия между ними невелики.

# Основные принципы
Основные элементы паттерна Flyweight включают:
- Интерфейс, определяющий методы, необходимые для реализации легковеса.
- Реализация интерфейса Flyweight, хранит внутренние состояния (stateless).
- Необобщаемые объекты, уникальные экземпляры которых нельзя повторно использовать.
- Класс, управляющий созданием и доступом к объектам Flyweight.
- Клиентский код, использующий интерфейс Flyweight для работы с объектами.

Ключевое различие между внутренними и внешними состояниями заключается в следующем:
- Внутреннее состояние — неизменяемые характеристики объекта, общие для всех пользователей.
- Внешнее состояние — переменные свойства объекта, зависящие от контекста конкретного использования.

# Преимущества
Использование паттерна Flyweight даёт ряд преимуществ:
- Значительная экономия оперативной памяти благодаря уменьшению числа создаваемых объектов.
- Улучшение производительности приложения путём снижения нагрузки на систему.
- Упрощение управления ресурсами системы при работе с большим количеством однотипных объектов.
Однако важно учитывать, что применение паттерна увеличивает сложность программы и требует тщательной организации внутренних состояний.

# Применение в реальных проектах
Паттерн Flyweight часто используется в приложениях, работающих с большими объёмами однородных данных, такими как графические редакторы, базы данных и текстовые процессоры. Например:
- В текстовых редакторах (например, Microsoft Word) для представления символов текста, где каждый символ имеет общие атрибуты (шрифт, размер шрифта), но разные значения.
- В игровых движках для отображения большого количества повторяющихся элементов (спрайтов, персонажей).
- В системах визуализации графики для уменьшения числа примитивов (точек, линий), представляющих сложную сцену.

Таким образом, паттерн Flyweight является мощным инструментом для оптимизации приложений, работающих с крупными объемами однородных данных, позволяя эффективно управлять памятью и улучшать производительность.
